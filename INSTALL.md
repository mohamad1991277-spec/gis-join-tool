# خطوات تثبيت وتشغيل البرنامج

1. **تثبيت بايثون:**
   لازم تنزل بايثون من الموقع الرسمي وتأكد إنك علمت على خيار "Add Python to PATH" وأنت بتثبته.

2. **تثبيت المكتبات:**
   افتح الشاشة السوداء (CMD) في مجلد المشروع واكتب الأمر هذا:
   ```bash
   pip install -r requirements.txt
   ```

3. **تشغيل البرنامج:**
   اكتب هذا الأمر في الـ CMD:
   ```bash
   streamlit run app.py
   ```
   بعدها البرنامج بيفتح معاك في المتصفح تلقائياً.

4. **توليد التقرير:**
   إذا تبي تطلع التقرير في ملف وورد، شغل الملف هذا:
   ```bash
   python generate_docx_ar.py
   ```
   بيطلع لك ملف وورد باسم GIS_Project_Report_Arabic.docx في نفس المجلد.
