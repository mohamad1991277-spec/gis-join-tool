# مشروع أداة الربط المكاني والوصفي - GIS
إعداد: [اسم الطالب]

هذا المشروع عبارة عن تطبيق ويب بسيط يساعد في دمج ملفات الـ GIS (الخرائط والبيانات المكانية) مع بعضها البعض سواء عن طريق الموقع الجغرافي أو عن طريق البيانات الوصفية.

## كيف تشغل البرنامج؟
1. لازم يكون عندك بايثون مثبت على الجهاز.
2. ثبت المكتبات المطلوبة باستخدام الأمر التالي:
   ```bash
   pip install -r requirements.txt
   ```
3. شغل البرنامج بالأمر التالي:
   ```bash
   streamlit run app.py
   ```

## أهم ميزات البرنامج
*   تقدر ترفع ملفات Shapefile (ملف مضغوط ZIP) أو ملفات GeoJSON.
*   يعرض لك الخريطة لكل ملف ترفعه عشان تتأكد من البيانات.
*   يسوي ربط مكاني (Spatial Join) بثمانية أنواع مختلفة.
*   يسوي ربط وصفي (Attribute Join) زي الجداول العادية.
*   تقدر تحمل النتيجة النهائية بصيغة GeoJSON.

## المكتبات اللي استخدمتها
استخدمت بايثون مع مجموعة مكتبات مشهورة في الـ GIS مثل:
- Streamlit (لواجهة الموقع)
- GeoPandas (للمعالجة الجغرافية)
- Folium (لعرض الخرائط)
- Shapely (للعمليات الهندسية)
